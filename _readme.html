Скрипт формы обратной связи PHP Mailer Final
Назначение

Практически на каждом сайте (и коммерческом, и развлекательном) необходима обратная связь с посетителями.

Необходимость такого скрипта:
  • Если опубликовать свой e-mail, то он быстро попадаем в базу спамеров и к Вам на ящик начинает идти непрерывный поток рекламы.
  • Посетителям намного удобнее прислать свое сообщение прямо с Вашего сайта (не совершая дополнительных действий).
Эти функции удобно реализовать с помощью данного скрипта, он выводит форму и, при заполнении необходимых полей, отправляет на Ваш e-mail сообщение. В скрипт встроена защита от роботов: перед отправкой письма надо ввести капчу.

Изменения в новой версии PHP Mailer Final

  • Купирована возможность взлома.
  • Изменен внешний вид формы. Упрощен внешний вид приходящего письма.
  • В капче отображаются цифры (раньше были цифры и буквы).

Крайне простая установка и настройки

  • Распакуйте содержимое архива на Ваш сайт.
  • Откройте текстовым редактором файл form/config.php.
  • Присвойте переменной $mailto = ""; свой e-mail.
    • Скрипт готов к работе.
    Протестируйте его работу, перейдя по ссылке http://ваш_сайт.ру/form/form.php

Лицензия

Пользоваться PHP Mailer Final можно абсолютно бесплатно при выполнении следующих условий:
  • Владельцам сайтов разрешается менять исходный код скрипта.
  • Не удаляйте, пожалуйста, линк внизу страницы на сайт автора.
  • PHP Mailer Final распространяется как есть ("as is").
